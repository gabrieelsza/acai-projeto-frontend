Açai Premium - Delivery
Este é um projeto de site responsivo de delivery focado em oferecer produtos de açaí, com interface moderna, carrinho de compras e navegação intuitiva.

Descrição
O projeto simula uma plataforma de venda e entrega de açaí (Aa Premium). O usuário pode navegar pelos produtos, adicionar itens ao carrinho e visualizar um resumo antes de finalizar o pedido. Conta também com informações institucionais da marca.

Estrutura dos Arquivos
index.html: Página principal com toda a estrutura do site, incluindo navegação, lista de produtos, carrinho, área institucional (Sobre) e contato.

style.css: Folha de estilo com layout moderno, responsivo, usando cores e tipografia agradáveis. Implementa as regras de estilização para sidebar, cards de produto, carrinho, botões, entre outros.

script.js: Scripts para navegação entre seções através de abas, além da função para adicionar produtos ao carrinho de compras de forma dinâmica e exibir alertas ao usuário.

Funcionalidades
Seção de produtos com diferentes tamanhos e sabores de açaí, além de opções especiais.

Sistema de carrinho de compras com quantidade e remoção de itens.

Informações institucionais, diferenciais da marca e missão.

Contato e links para redes sociais.

Layout adaptado para dispositivos móveis.

Como Usar
Clone ou faça download dos arquivos do repositório.

Abra o arquivo index.html em seu navegador.

Navegue pelas abas de produtos, adicione itens ao carrinho e explore as funcionalidades.

Requisitos
Navegador moderno (Google Chrome, Firefox, Edge, etc.).

Não são necessárias dependências externas para rodar o projeto. Todo o funcionamento é realizado em HTML, CSS e JavaScript puro.

Observações
As imagens de produtos são ilustrativas (placeholders).
